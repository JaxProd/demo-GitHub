git add 'nom_fichier' : 
ajoute ton fichier à la liste des fichiers à la liste à surveiller

git add . : 
ajoute tous les fichiers au pool de push


git status : 
analyse les modifications de fichiers

git branch : 
lister les branches
* nom désigne la branche actuelle

Créer une nouvelle branche :
git branch nom-de-branche 

git checkout nom_de_la_branche : 
permet de basculer d'une branche à l'autre

Fusionner une branche sur la branche actuelle : 
git merge nom_de_la_branche_a_fusionner_avec_le_master

Supprimer une branche : 
git branch -d nom_de_la_branche_a_supprimer

Afficher la liste des commit et la référence :
git loglangue

git commit -m "text de description"
Crée une nouvelle version

-------------------------------------------------

git push origin master : 
upload le projet sur GitHub


Enregistrer les identifiants GitHub
git config --global user.email "em@ail.com"
git config --global user.name "Identifiant"
git config --global user.password "IdentMotDePasse"


Vérifier la configuration enregistrée : 
git config --list
